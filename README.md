# SyPastebin
An online light Pastebin for developers.

一个前后端分离的在线剪贴板，提供在线粘贴代码、生成可供访问的url的功能。

# Usage
1. springboot 2.4.2
2. maven 3.6.3
3. mysql 8.0
4. mongodb 4.4.3
5. JDK 1.8
